# 博客绑定二级域名

### 背景

已经有了 Github Pages 绑定个人主域名，想要将其他的仓库绑定到次级域名。

### 实践

**云解析**

云解析上面添加一条新的解析指令即可。

![image-20200602164046331](博客绑定二级域名/image-20200602164046331.png)

**Github**

Github 上面 配置 `Custom Domain` 

Github 会自动进行激活 及 相应的认证操作

认证通过后，就可以激活 `HTTPS` 了
